## ✨ Features

- **🎨 Modern Glassmorphism UI**: A sleek, frosted-glass design aesthetic using vanilla CSS.
- **📱 Fully Responsive**: Seamlessly adapts to smartphones, tablets, and desktop displays.
- **✨ Immersive Animations**: Scroll reveals, background particle effects, and floating gradient orbs.
- **🌙 Deep Dark Mode**: Optimized color palette for eye comfort with neon accent highlights.
- **🖼️ Lightbox Image Gallery**: Integrated modal to gracefully view achievements and hall of fame captures.

## 🛠️ Built With

- **[HTML5](https://developer.mozilla.org/en-US/docs/Web/HTML)** - Semantic structure
- **[CSS3](https://developer.mozilla.org/en-US/docs/Web/CSS)** - Styling, Animations, and Layouts (`style-modern.css`)
- **[JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)** - Interactive components & modals (`custom.js`)
- **[Bootstrap](https://getbootstrap.com/)** - Grid system and responsiveness
- **[jQuery / FullPage.js](https://alvarotrigo.com/fullPage/)** - Smooth scrolling section behaviors
- **[Animate.css](https://animate.style/) & [WOW.js](https://wowjs.uk/)** - Scroll reveal animations

## 🚀 Quick Start (Local Setup)

Want to run this locally on your machine? Follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nanuzn/Portfolio2.git
   ```
2. **Navigate into the directory:**
   ```bash
   cd Portfolio2
   ```
3. **Open `index.html`** in your favorite browser! (Or simply use an extension like Live Server in VS Code).
